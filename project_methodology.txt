AI-Based Country Future Demographic Prediction System

Methodology

1. Data Collection
- The project uses a World Bank-style CSV dataset containing country-wise demographic data.
- The dataset includes yearly values for multiple countries.

2. Data Cleaning
- Aggregate entries such as World, regions, and income groups are removed.
- Only real country-level records are kept.
- Missing values represented by ".." or blank cells are converted to missing values and interpolated.

3. Data Reshaping
- The original dataset is in wide format, where each year is a separate column.
- It is converted into long format and then into a country-year table.
- Final columns used in the model include:
  - Year
  - Male
  - Female
  - Urban
  - RuralPercent
  - BirthRate
  - DeathRate

4. Feature Engineering
- TotalPopulation is calculated as Male + Female.
- RuralPopulation is calculated as TotalPopulation multiplied by RuralPercent / 100.
- These derived features help complete the demographic forecast.

5. Machine Learning Model
- The project uses supervised learning with Linear Regression.
- Input (X): Year
- Output (Y): demographic value
- Separate models are trained for:
  - Total Population
  - Birth Rate
  - Death Rate
  - Urban Population

6. Prediction Logic
- The model predicts demographic values for a selected future year.
- Male, Female, and Rural Population are not trained directly.
- They are estimated from historical ratios.

7. Time Series Projection
- The system can generate a full projection from the last available year to the selected future year.
- This allows the charts to show trends up to 2100.

8. Visualization
- The Streamlit dashboard displays:
  - Population trend line chart
  - Birth and death rate chart
  - Male vs female bar chart
  - Urban vs rural pie chart
  - Prediction cards and processed data table

How to explain it to a teacher

"This project is a demographic forecasting system built with Python and Streamlit. I used historical country-wise demographic data, cleaned and reshaped it, then trained Linear Regression models where the input was the year and the output was demographic variables like population, birth rate, death rate, and urban population. The app then predicts future values for any selected country and year, and displays the results in charts and tables."
